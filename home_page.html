Parameterized Thread Start Delegate

Working with the ParameterizedThreadStart Delegate Recall that the ThreadStart delegate can point only to methods that return void and take no arguments. While this might fit the bill in some cases, if you want to pass data to the method executing on the secondary thread, you will need to use the ParameterizedThreadStart delegate type.
